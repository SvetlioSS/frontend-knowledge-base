# HTML APIs

Full list available here https://developer.mozilla.org/en-US/docs/Web/API

## Location & Device APIs
* <strong>Geolocation API</strong> – Get user’s location.
* <strong>Device Orientation API</strong> – Access accelerometer and gyroscope data.
* <strong>Device Motion API</strong> – Detect physical movement of the device.
* <strong>Battery Status API</strong> – Get battery charge and status info (deprecated in most browsers for privacy).

## Media & Hardware Access APIs
* <strong>MediaDevices API</strong> – Access camera, microphone, screen (e.g., getUserMedia()).
* <strong>WebRTC API</strong> – Real-time audio, video, and data communication.
* <strong>Screen Capture API</strong> – Capture screen/window for recording or sharing.

## File & Storage APIs
* <strong>File API</strong> – Read and manipulate files from file inputs or drag-and-drop.
* <strong>File System Access API</strong> – Read and write to the user's local file system.
* <strong>Web Storage API</strong> – localStorage and sessionStorage.
* <strong>IndexedDB API</strong> – NoSQL-style database for storing large amounts of structured data.
* <strong>Cache API</strong> – Store and retrieve network requests and responses.
* <strong>Cookies</strong>

## Networking & Communication APIs
* <strong>Fetch API</strong> – Modern way to make HTTP requests.
* <strong>WebSocket API</strong> – Bi-directional communication with a server.
* <strong>Server-Sent Events (EventSource)</strong> – Push updates from server to client.
* <strong>Beacon API</strong> – Send analytics/data to server asynchronously when page is unloading.

## Web Workers & Threads
* <strong>Web Workers API</strong> – Run scripts in background threads.
* <strong>Service Workers API</strong> – Control caching, offline access, and background sync.

## User Interaction APIs
* <strong>Drag and Drop API</strong> – Built-in drag-and-drop support.
* <strong>Clipboard API</strong> – Read from and write to the system clipboard.
* <strong>Fullscreen API</strong> – Display elements in full screen.
* <strong>Vibration API</strong> – Make the device vibrate (mostly for mobile, limited support).
* <strong>Gamepad API</strong> – Access game controller input.

## AI & Sensor APIs
* <strong>Speech Recognition API</strong> – Convert spoken input to text.
* <strong>Speech Synthesis API</strong> – Convert text to spoken voice.
* <strong>WebXR / WebAR / WebVR APIs</strong> – Support for virtual and augmented reality experiences.
* <strong>Ambient Light Sensor API</strong> – Detect ambient light level (limited support).

## Graphics & Animation APIs
* <strong>Canvas API</strong> – Draw 2D graphics.
* <strong>WebGL / WebGL2</strong> – Render 3D graphics.
* <strong>Web Animations API</strong> – Control animations using JavaScript.
* <strong>SVG API</strong> – Scalable vector graphics manipulation.

## Other Useful APIs
* <strong>Notifications API</strong> – Show system notifications.
* <strong>Permissions API</strong> – Check and request permissions for APIs (e.g., geolocation).
* <strong>Intersection Observer API</strong> – Efficiently watch element visibility (e.g., lazy-loading).
* <strong>Mutation Observer API</strong> – Watch for DOM changes.
* <strong>Resize Observer API</strong> – Watch for size changes of elements.
* <strong>Credential Management API</strong> – Manage user credentials (passwords, federated login).
* <strong>Payment Request API</strong> – Integrate with native payment systems.
* <strong>Web Share API</strong> – Share content via native share dialog (mobile-first).
